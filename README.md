# Dysarthric-Speech-Communication-Aid
An assistive communication system for dysarthric speech that combines Whisper ASR, FLAN-T5-Large for text correction, and TTS. It converts impaired speech into accurate text and natural-sounding voice output, improving communication and accessibility for users with speech disorders.
